# Rspress website

新版个人博客网站施工中 🛠️

## Setup

Install the dependencies:

```bash
bun install
```

## Get started

Start the dev server:

```bash
bun run dev
```

Build the website for production:

```bash
bun run build
```

Preview the production build locally:

```bash
bun run preview
```
